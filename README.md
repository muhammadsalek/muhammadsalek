> **⚡ The snake appears after one Action run.** Follow the 3 steps below — takes under 2 minutes.

<details>
<summary><b>🛠️ Click here — 3-step snake setup</b></summary>

<br/>

**STEP 1 — Create the file** `.github/workflows/snake.yml` in your `muhammadsalek/muhammadsalek` repo and paste:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"    # daily at midnight UTC
  workflow_dispatch:         # manual trigger button

permissions:
  contents: write

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:

      - name: Generate snake SVGs
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: muhammadsalek
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Deploy to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**STEP 2 — Allow Actions write permission**
Repo → **Settings** → **Actions** → **General** → **Workflow permissions** → select **"Read and write permissions"** → Save.

**STEP 3 — Run it**
**Actions tab** → **Generate Snake Animation** → **Run workflow** → wait ~60 s → ✅ snake appears.

The snake now regenerates automatically every day at midnight UTC.

</details>
